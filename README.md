本代码用于验证谐波源定位问题中所构建的**比例指纹（Proportional Fingerprint）**的有效性。
在低可观测条件下，部分节点由于测量缺失或网络耦合关系，难以仅依靠传统幅值/相角响应实现准确区分。为此，本代码通过对节点响应进行比例化处理，构建与注入强度和整体相位偏移相对解耦的特征表示，用于分析：

比例指纹对不同候选节点的区分能力；
比例指纹对边界节点与内部节点的辨识效果；
比例指纹与物理指纹在欠定区域中的响应差异；
比例指纹在多谐波联合条件下的稳定性与可分性。

本代码主要服务于“低可观测条件下谐波源定位”方法的验证环节，可作为后续比例空间稀疏匹配、区域判别和精细定位的基础。
